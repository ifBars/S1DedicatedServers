# Changelog

## 0.9.0-beta - 2026-04-12

- Add initial IL2CPP client and server support across connection lifecycle hooks, world initialization, and save bootstrap flows.
- Reorganize the public mod API surface around focused client, server, and metadata helpers while keeping `S1DS` as the coarse entry point.
- Improve dedicated-server runtime behavior with lower headless audio and visual overhead, pathing optimizations, ghost-host stat cleanup, and updated shutdown messaging.
- Default new server configs to `SteamGameServer` authentication with a `30` second auth timeout, and refresh the configuration docs accordingly.
- Refresh release-facing documentation for IL2CPP support, including guidance to report IL2CPP issues on the GitHub issue tracker.
